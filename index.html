Sobre Mim

Sou Renan Fumis, Desenvolvedor Web Full Stack com maior ênfase e domínio em tecnologias de Frontend. Minha formação inicial é em Publicidade e Propaganda, o que contribuiu para minha visão estética e atenção à experiência do usuário — mas foi no contato com equipes de desenvolvimento em empresas de tecnologia que descobri minha real vocação: programar.

Sou pós-graduado em Desenvolvimento Full Stack pela PUC-Minas e venho trabalhando com HTML, CSS (Grid/Flex), JavaScript e React, criando interfaces modernas, acessíveis e responsivas. No backend, atuo com Python (Django) e Node.js (Express), desenvolvendo APIs RESTful, integrações com serviços externos e soluções completas para web.
